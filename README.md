# To-Do List Management System

## Description
This project is a simple To-Do List Management System developed using C++ and Object-Oriented Programming concepts.

## OOP Concepts Used
- Classes
- Objects
- Encapsulation
- Inheritance
- Functions

## Features
- Add Task
- View Tasks
- Mark Task as Complete
- Exit

## Debugging
An intentional indexing bug was introduced and then fixed. The debugging process is explained in the Debugging Report.
